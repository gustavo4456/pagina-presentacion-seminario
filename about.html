Proyecto realizado para la cátedra de seminario de actualización ii

Tecnologias selecciondas

Nosotros seleccionamos las tecnologias de react native para nuestro frontend y spring boot para el backend.

Otras tecnologias

Axios, heroku, android studio, node.js, jdk, expo go.

Despliegue

Utilizamos la nube de heroku para subir la api y así poder realizar pruebas desde emuladores o desde nuestros propies dispotivos, esto nos permitido compartir generar una apk para que amigos, familiares y nosotros mismos podamos probar la app en nuestros celulares.

Aplicacion de ejemplo

Para reforzar todo lo aprendido en nuestra investigación tanto en frontend (react native) como en el backend (Spring boot) decidimos crear una apliación que nos permite crear un usuario, iniciar sesión, enviar solicitudes de amistad, cancelarlas, rechazarlas, eliminar amigo, crear editar y eliminar publicaciones (foto, titulo y contenido), y compartir con amigos. También desarrollamos otra app utilizando expo lo cual nos facilito mucho las cosas, ya que es mucho mas simple correr un proyecto con expo.
